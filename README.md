#uzplayer

这其实是一个kxmovie的分支，不知道为何，大家在我的blog中反应直接从master中下载得到的kxmovie已经
不能进行编译了(文章地址http://blog.csdn.net/zengraoli/article/details/13004701)。
所以我尽量以我自己的时间，维护一个kxmovie分支。


后期，将要添加一些比如手势的、连接网络的、p2p播放的功能。


感谢kxmovie的作者，如果没有这个项目，至少uzplayer会陷入一个麻烦的境地，我本人不善于做gui，至少设计得不大像样！
